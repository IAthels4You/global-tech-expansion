PROJECT CONTEXT: GLOBAL TECH HUB EXPANSION
==========================================
Role: Senior Analytics Engineer
Objective: Automate a professional E2E data pipeline to calculate the "Global Expansion Opportunity Index".

1. TECH STACK
-------------
- Data Warehouse: Snowflake (FinOps optimized).
- Orchestration: GitHub Actions (CI/CD).
- Transformation: dbt Core (Medallion Architecture).
- Languages: Python & SQL.
- Data Source: World Bank API (GDP & Inflation) + Talent/Salary Benchmarks.

2. DATA ARCHITECTURE (MEDALLION)
--------------------------------
- RAW (Bronze): Landing zone for raw API data (JSON/CSV).
- TRANSFORM (Silver): Staging, Data Contracts (Pydantic), and Intermediate models.
- ANALYTICS (Gold): Final OBT (One Big Table) for BI consumption.

3. IMPLEMENTATION PHASES
------------------------
PHASE I: Ingestion & Validation
- Automate Python scripts for World Bank API.
- Implement Data Contracts with Pydantic.
- Setup GitHub Actions for CI/CD.

PHASE II: Cloud Infrastructure (Snowflake)
- Setup Database: GLOBAL_EXPANSION.
- Setup Schemas: RAW, TRANSFORM, ANALYTICS.
- Setup Warehouse: X-SMALL with 60s auto-suspend (FinOps).

PHASE III: dbt Modeling & Governance
- ISO 3166-1 Alpha-3 standardization.
- Implementation of dbt Snapshots (SCD Type 2) for historical tracking.
- Automated dbt tests (Uniqueness, Not Null, Singular tests).

PHASE IV: OBT & Visualization
- Table: fct_global_tech_matrix (Denormalized).
- Metrics: Weighted Score (40% Stability, 40% Talent, 20% Cost).

PHASE V: Forecasting (Snowpark)
- Use Snowpark Python for GDP/Inflation projections (Prophet).
- Talent growth forecasting (10-20 years horizon).

4. KEY KPIs
-----------
- Global Expansion Opportunity Index.
- Talent/Cost Ratio.
- Political Stability & Devaluation Trends.
